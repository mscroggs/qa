UnitCircle was removed altogether, so you will need to use the mshr component to make a circular mesh.

answered Oct 20, 2015 by chris_richardson FEniCS Expert (31,740 points)
0 votes

Adding to the answer above:

This is what you need:

domain = mshr.Circle(Point(0.,0.),1.0,60)
mesh = mshr.generate_mesh(domain, 60, "cgal")
