I was wondering if there is some external software for generating a mesh. I would like to save an externally generated mesh to a file, so that I can afterwards access and work on the saved externally generated mesh from within a FEniCS program.

People usually use gmsh - http://geuz.org/gmsh/ - there are conversion tools in DOLFIN to convert to the DOLFIN format.
It is also possible to save meshes from ParaView in XDMF format, which is useful sometimes.
