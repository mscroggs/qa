If u is your vector you can do

N = u.vector.size()
u.vector()[:] = numpy.random.random(N)

Be aware that the mean will then be 0 in l1 rather than L1.
If you want it to be zero in L1 you would do something
like

c = assemble(u*dx)
u.vector()[:] -= c
