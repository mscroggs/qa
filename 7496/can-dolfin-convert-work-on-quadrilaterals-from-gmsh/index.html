Can dolfin-convert work on quadrilaterals from GMSH?

0 votes

If I understand FEniCS correctly, it can support both triangular and quadrilateral elements. However, when I use dolfin-convert on a quadrilateral mesh, it seems to produce a mesh containing only the exterior edge nodes. I saw another thread from about half a year ago where someone said quad elements were not yet supported. Is this still true today?

  • dolfin-convert
  • gmsh

Quadrilaterals and hexahedra are not supported in current versions of DOLFIN.
In release 1.6 there will be experimental support for quad and hex mesh i/o, but no support for assembly yet. Full support will not be included before release 1.7
