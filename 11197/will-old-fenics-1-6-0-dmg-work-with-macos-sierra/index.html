No, we have no plans to release any more dmg packages for macOS. Someone is looking into making binary packages available for Anaconda, but there is no timeline for that. Your option for now is to build from source, either manually or automatically using the fenics-install.sh script. I expect FEniCS to work on macOS Sierra, but I haven't tested yet.
