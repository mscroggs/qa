Hi to everybody.
I have to solve a nonlinear problem and to start I have modified the demo_nonlinear_poisson.
Actually in my problem the nonlinearity is in the neumann bc which depends on the solution and on the space. The dependence on the space involves the % (mod) function. Apparently Fenics is not able to compute the jacobian when the mod function is present in the formulation. Is it right ? Is there any workaround ?

UFL does not have any mod or % function altough it can defined using conditional operator. You have to define your forms using operators supported by UFL.

On the other hand mod(x, y) is not differentiable function in both arguments. When it would be defined through conditional it would throw a warning similiar to Differentiating conditional, assuming continuity, condition will not be differentiated. Derivative of discontinuity would not be present and as a result Newton method would not possess quadratic convergence (if any convergence at all) when supposed to cross over discontinuity.

I realized that floor, ceiling or round operator would be needed to implement modulo using conditional. These operators are not available in UFL but maybe there is some trick.

Last solution left is to define something like

class mod(Expression):
     def __init__(self, u, v):
         self.u = u
         self.v = v
     def eval(self, values, x):
         values[0] = self.u(x)%self.v(x)
     def eval_cell(self, values, x, cell):
         u, v = numpy.zeros(1), numpy.zeros(1)
         u = self.u.eval_cell(u, x, cell)
         v = self.v.eval_cell(v, x, cell)
         values[0] = u%v
     def update(self):
         self.u.update()
         self.v.update()
     def ufl_element():
         elements = [self.u.ufl_element(), self.v.ufl_element()]
         return max((element.degree(), element) for element in elements)[1]

and use this to define modulo operation mod(u, v) for Functions u, v which can be used in forms. But you can't use it for form Arguments, Trial/TestFunctions and differentiate it. Hence it can be useful for fixed-point iteration.
