Not exactly. I was hoping for some kind of vectorized operation, without any explicit for looping. Right now I'm doing, in 1D:

xx = np.linspace(0, 1, num=100)
uu = np.zeros_like(xx)

for i in np.arange(0, xx.size):
    uu[i] = u(xx[i])
