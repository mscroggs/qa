Best answer

yes,

from dolfin import *
mesh = RectangleMesh(Point(0.,0.),Point(2.,1.), 20,10)
from pylab import show, triplot
coords = mesh.coordinates()
triplot(coords[:,0], coords[:,1], triangles=mesh.cells())
show()

works great in 2D.

For 3D I would suggest paraview.
